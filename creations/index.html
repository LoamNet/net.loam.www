Games


  • Line Zen

    Android | Relaxing 2D Puzzle Game
    Closed Beta

    This mellow game is about collecting bubbles by drawing lines. It's designed to be a relaxing spatial puzzler, providing a mix of handmade and generated challenges!



  • Bytewalker

    Android | Tile Based Strategy Game
    Open Alpha

    Inspired by older tile dungeons, this 2D tile based strategy game is about waging digital warfare in the far future.






Apps and Tools


  • Preview MD

    Web App | A markdown preview tool
    Released v1.0

    This website uses a good bit of markdown for things like the blog posts, and having a tool to help preview how that looks is useful - you can get a feel for how your markdown looks when parsed and displayed as if it were a post on this site too!







Experiments


  • Salmonberry

    Web, PC | 2D Multiplayer Roguelite-ish
    Open Experiment

    Salmonberry is an experimental multiplayer roguelite drawing from classics like Brogue, but with a more social and casual feel to it. The game has mutliple clients across different platforms with different visual styles.



  • Thoughtful Snake

    TBD | 2D tile-based strategy game
    Closed Experiment

    Thoughtful Snake is a slower paced take on the classic Snake genre of games! It's experimenting with bringing more strategy and consideration to the table than the real-time original.



  • Space Candy

    TBD | Idle simulation game
    Closed Experiment

    A space-themed idle management game for Android! This is exploring constructing and commanding your ship and crew, making your mark among the stars.
